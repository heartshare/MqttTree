MQTT：
           是轻量级基于代理的发布/订阅的消息传输协议，设计思想是开放，简单，易于实现。
           
           特点：
               1）使用发布/订阅消息模式，提供一对多的消息发布，解除应用程序耦合。
               2）对负载内容屏蔽的消息传输。
               3）使用TCP/IP提供网络连接。
               5）小型传输，开销很小，协议交换最小化，以降低网络流量。
